Locator help

1. Search field

There are a total of three input fields on the page.

First field 'Search' used for normal locator searches. Second and third fields 'Locator A' and 'Locator B' which are used to measure two locators.

2. Search results

In the case of a normal search for a locator, the given locator is displayed on the map as a result, which is bordered by a black color.

The locator measurements are shown on the map as symbols A (red) and B (blue), and a green line is shown between them to show a direct route between the two points. The distance between points A and B will also be displayed below the A input field.

3. Other functions

To search and measure locators, it is possible to use various functions that will make the search more pleasant:

*A marker with your location will appear on the map
The locator and coordinates will be displayed
*Adds the current locator to the Locator A search field
*Adds the current locator to the Locator B search field
Adds the locator after clicking the mouse in the Locator A search field
Adds the locator after clicking the mouse in the Locator B search field
Displaying the locator with the mouse (top right)
* Access to your location is required to use this feature.
The feature works best on mobile, but the location display should be fine on PC as well.

4. QSO map

There are two input forms available on the page. The upper one is for entering the Main locator (your locator), the lower one is used for entering Multiple locators (counter stations locator). The result is the displayed connections that are connected to the main locator. After clicking on a point on the map, the given locator and the distance from the main locator will be displayed.

*Adds the current locator to the Main locator field
Displays a classic map layer
* Access to your location is required to use this feature.
The feature works best on mobile, but the location display should be fine on PC as well.

After displaying the result, the number of contact (QSO), the main locator (QTH) and also the longest contact (QRB) will be displayed in the map at the bottom left.

5. Maps

A marker with your location will appear on the map
The locator and coordinates will be displayed
Displaying the coordinates and locator with mouse (top)
* Access to your location is required to use this feature.
The feature works best on mobile, but the location display should be fine on PC as well.
